# agentic-swe-site

Public documentation and marketing site for [**agentic-swe**](https://github.com/agentic-swe/agentic-swe), built with Vite + React.

- **Live site:** https://agentic-swe.github.io/agentic-swe-site/
- **Source pack / plugin:** https://github.com/agentic-swe/agentic-swe

## Local development

```bash
npm ci
npm run dev
```

Production build uses `VITE_BASE=/<repo>/` in GitHub Actions for GitHub Pages.
